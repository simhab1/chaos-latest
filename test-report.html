10. Installation Instructions

1. Open Chrome and go to chrome://extensions/ 2. Enable "Developer mode" in top right 3. Click "Load unpacked" 4. Select the ChaosTraderX folder containing manifest.json 5. Extension should appear in toolbar 6. Navigate to Exness platform 7. Click extension icon to open control panel 8. Configure settings and test functionality

Final Note: This extension represents a sophisticated automated trading system. Always test on demo accounts first and understand the risks of automated trading before using on live accounts.
